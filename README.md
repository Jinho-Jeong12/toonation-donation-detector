# 투네이션 후원 감지기

> **본 프로그램은 법적 증거 보존을 목적으로 제작되었습니다.**
> 방송 영상에서 투네이션 후원 내역을 자동으로 감지하여 스크린샷, CSV, Excel 파일로 저장합니다.

## 📥 프로그램 다운로드 (GitHub에서 설치하기)

### 1. GitHub 페이지 접속

아래 링크를 클릭하거나 브라우저 주소창에 붙여넣으세요.

```
https://github.com/Jinho-Jeong12/toonation-donation-detector
```

### 2. 파일 전체 다운로드

1. 페이지 오른쪽 위 초록색 **`<> Code`** 버튼 클릭
2. **`Download ZIP`** 클릭
3. 다운로드된 ZIP 파일을 원하는 폴더에 **압축 해제**

> **Git이 설치된 경우** 아래 명령어로도 받을 수 있습니다.
> ```
> git clone https://github.com/Jinho-Jeong12/toonation-donation-detector.git
> ```

### 3. 압축 해제 후 폴더 내용 확인

압축을 풀면 아래 파일들이 있어야 합니다.

| 파일 | 설명 |
|------|------|
| `extract_toonation.py` | 투네이션 후원 감지 프로그램 |
| `투네이션 1,000원 효과음.mp3` | 음성 감지용 효과음 (필수) |
| `투네이션 10,000원 후원 효과음.mp3` | 음성 감지용 효과음 (필수) |
| `README.md` | 사용 설명서 (이 파일) |

> **mp3 파일 2개는 반드시 `extract_toonation.py`와 같은 폴더에 있어야 합니다.**
> 이 프로그램은 방송 영상의 오디오에서 투네이션 후원 효과음을 찾아낼 때, 이 mp3 파일들을 **"기준 소리"** 로 사용합니다.
> mp3 파일이 없으면 음성 감지(B조건)가 비활성화되어 화면 텍스트 인식(A조건)만으로 동작하므로 일부 후원을 놓칠 수 있습니다.

## ⚠ 실행 전 필수 — Python 설치

**Python이 반드시 설치돼 있어야 합니다.**
Python만 있으면 나머지(scipy, ffmpeg 등)는 프로그램 실행 시 자동으로 설치됩니다.

### Python 설치 방법
1. https://www.python.org/downloads/ 접속
2. **Download Python 3.x.x** 버튼 클릭
3. 설치 시 **반드시 "Add Python to PATH" 체크** 후 설치

## 🚀 사용법

### 1. `extract_toonation.py` 실행

#### 실행 방법 (Windows)
- 파일을 **더블클릭** 하거나, 터미널에서 아래 명령어 실행
  ```
  python extract_toonation.py
  ```

#### 실행 방법 (macOS)
1. `extract_toonation.py` 파일을 **우클릭(오른쪽 클릭)** 합니다.
2. **"다음으로 열기"** → **"IDLE"** 을 선택합니다.
3. IDLE 창이 열리면 상단 메뉴에서 **Run → Run Module** 을 클릭합니다. (단축키: **F5**)

---

- 실행 후 화면이 뜨기까지 **10초 정도** 걸릴 수 있습니다. 기다려 주세요.
- 처음 실행 시 필요한 패키지와 ffmpeg를 자동으로 다운로드합니다 (수 분 소요)

### 2. 영상 파일 선택
- 파일 선택 창이 열리면 분석할 방송 영상 파일 선택 (mp4, mkv, webm 등)

### 3. 자동 분석 시작
- 프로그램이 영상을 분석하며 투네이션 후원을 감지합니다
- 터미널에 진행 상황이 실시간으로 표시됩니다

### 4. 결과 확인
- 영상과 같은 이름의 폴더가 생성됩니다: `영상이름_스크린샷/`
- 후원 감지될 때마다 해당 시점의 **방송 화면 스크린샷**이 저장됩니다
- 추가로 Excel(`.xlsx`)과 CSV(`.csv`) 파일도 함께 저장됩니다

## 🔍 감지 방식

후원을 두 가지 방법으로 동시에 감지합니다 (둘 중 하나라도 감지되면 저장).

| 조건 | 방법 | 설명 |
|------|------|------|
| **A조건** | 화면 텍스트 인식 | 파란 글씨 "닉네임 님이 N캐시", "영상 후원" 등 패턴 |
| **B조건** | 음성 효과음 인식 | 1,000원 / 10,000원 투네이션 효과음 매칭 |

> 프로그램이 A 또는 B 중 하나를 놓치더라도 나머지 조건으로 보완됩니다.

## ⚙ 자동 설치 목록

프로그램 첫 실행 시 아래 항목이 자동으로 설치됩니다.

- `scipy` — 음성 파형 분석
- `openpyxl` — Excel 파일 생성
- `easyocr` — 화면 텍스트 인식 (한국어)
- `opencv-python` — 영상 프레임 처리
- `ffmpeg` — 영상에서 오디오 추출 (스크립트 옆 `ffmpeg_bin/` 폴더에 저장)

## ❓ 자주 묻는 질문

**Q. ffmpeg 다운로드에 실패했다고 뜹니다.**
A. 인터넷 연결을 확인하세요. ffmpeg가 없으면 음성 감지(B조건)는 비활성화되고 화면 텍스트 인식(A조건)만으로 동작합니다. 일부 후원을 놓칠 수 있습니다.

**Q. 분석 중간에 꺼도 되나요?**
A. 네. 진행 상황이 자동 저장되므로 다시 실행하면 중단된 지점부터 재개됩니다.
